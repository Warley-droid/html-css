Hyperlinks

Links externos

Links externos são elementos que permitem navegar entre páginas da web, para adicionarmos um link externo utilizamos a tag <a> e adicionamos o link na tag interna de <href>=" "

Se o Hyperlink nos direcionar para uma página que está escrita em outro idioma devemos adicionar a tag interna <lang>

Para nos indicar o local onde o novo link será aberto, utilizamos a tag interna <target>

Para nos indoicar a origem do link, utilizamos a tag interna <rel>

Links internos são elementos que permitem navegar entre páginas da mesma web, para adicionarmos um link interno utilizamos a tag <a> e adicionamos o link na tag interna de <href>=" "

Links para downloads

Para adicionar um link para download, utilizamos a tag <a> e adicionamos o link na tag interna de <href>=" ", adicionamos também a tag interna <download> e por útimo adicionams a tag interna <type> para adicionar o tipo de arquivo. Exemplo de um link para download:
