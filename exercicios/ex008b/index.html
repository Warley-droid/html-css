Tipos de formatações diferentes

Código-Fonte

Serve para deixar o espaçamento entre letras iguais, ou seja, o texto fica com a formatação de código-fonte, como se fosse um código de programação. Exmeplo: console.log("Olá, mundo!"); Quando o código tem mais de uma linha pode ser usado a tag <pre> para manter a foermataçao original do texto, como no exemplo abaixo: 

            
                function saudacao() {
                    console.log("Olá, mundo!");
                }

Citações

Como o próprio nome diz, a tag <q> serve para indicar uma citação, ela é ideal para usar em pequenas citações, sua principal característica é as aspas, como no exemplo atrás como diz o Geremias: O importante é a saúde, o resto é besteira.

Citações completas

Para citações mais longas, pode-se usar a tag <blockquote>, que adiciona uma indentação e formatação específica para destacar o trecho citado, como no exemplo abaixo:

"O importante é a saúde, o resto é besteira."

Geremias
Podemos também usar a extensão da tag cite para indicar a fonte da citação, como no exemplo abaixo:

"O importante é a saúde, o resto é besteira."

De trás para frente

A tag <bdo> (bidirectional override) é usada para controlar a direção do texto, permitindo que ele seja exibido da direita para a esquerda (RTL) ou da esquerda para a direita (LTR), como no exemplo abaixo: Este texto será exibido da direita para a esquerda.
